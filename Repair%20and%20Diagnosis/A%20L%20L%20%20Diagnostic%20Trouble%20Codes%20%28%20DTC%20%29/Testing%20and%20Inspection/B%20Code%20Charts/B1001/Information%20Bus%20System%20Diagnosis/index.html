Information Bus System Diagnosis



DTC B1001

DTC DESCRIPTOR

DTC B1001 00
Option Configuration Error

DTC B1001 45
Option Configuration Error Variant Not Programmed

DTC B1001 4B
Option Configuration Error Calibration Not Learned

DIAGNOSTIC INSTRUCTIONS
- Perform the Diagnostic System Check - Vehicle prior to using this diagnostic procedure Initial Inspection and Diagnostic Overview.
- Review Strategy Based Diagnosis for an overview of the diagnostic approach.
- Diagnostic Procedure Instructions provides an overview of each diagnostic category.

CIRCUIT/SYSTEM DESCRIPTION
The internal fault detection is handled inside the control module. No external circuits are involved.

CONDITIONS FOR RUNNING THE DTC
The only requirements are voltage and ground.

CONDITIONS FOR SETTING THE DTC

DTC B1001 00
The module has detected that the options are not programmed correctly.

DTC B1001 45
The module has detected that the restraints ID number is not programmed.

DTC B1001 4B
The module has detected that the axle/tire calibration is not programmed.

CONDITIONS FOR CLEARING THE DTC
- A current DTC clears when the condition is no longer present.
- A history DTC clears when the module ignition cycle counter reaches the reset threshold, without a repeat of the malfunction.

CIRCUIT/SYSTEM VERIFICATION

DTC B1001 00
Program the body control module (BCM). Refer to Service Programming System (SPS).
- If the DTC remains current, replace the BCM.

DTC B1001 45
1. Program the restraints ID into the BCM.
2. Retrieve DTCs from the BCM. DTC B1001 45 should change to a history DTC.
- If the DTC remains current, replace the BCM.

DTC B1001 4B
1. Program the tire/axle RPO into the BCM.
2. Retrieve DTCs from the BCM. DTC B1001 4B should change to a history DTC.
- If the DTC remains current, replace the BCM.
