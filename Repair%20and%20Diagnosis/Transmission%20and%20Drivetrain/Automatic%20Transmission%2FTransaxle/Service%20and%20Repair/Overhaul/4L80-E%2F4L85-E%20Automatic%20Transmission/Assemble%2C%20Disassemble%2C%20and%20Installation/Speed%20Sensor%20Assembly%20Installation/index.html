Speed Sensor Assembly Installation



Speed Sensor Assembly Installation




1. Install the input and output speed sensors (22), (output speed sensor plug-4WD) into the transmission case.

Notice: Refer to Fastener Notice.

2. Install the input and output speed sensor bolts (23).

Tighten the bolts to 11 N.m (97 lb in).
