Scan Tool Data Definitions


Scan Tool Data Definitions

ASC Disable Switch:This parameter displays the state of the ASC switch it will be active or inactive

ASC Learn Status:This parameter displays the learned status of the ASC module it will display running, passed or cancelled.

ASC System Status:This parameter displays the status of the ASC system it will display unknown, normal, disabled or locked.

Battery Voltage:This parameter displays the battery voltage at the ASC module.

Driver Door Ajar Sw.:This parameter displays the status of the driver door ajar switch it will display door closed or door ajar.

Passenger Door Ajar Sw:This parameter displays the status of the passenger door ajar switch it will display door closed or door ajar.

Left Rear Door Ajar Sw:This parameter displays the status of the left rear door ajar switch it will display door closed or door ajar.

Right Rear Door Ajar Sw:This parameter displays the status of the right rear door ajar switch it will display door closed or door ajar.

Left ASC Board Commanded:This parameter displays the commanded position of the left running board.

Left ASC Board Learned:This parameter will display yes or no to let you know if it has learned it's fully extended and fully retracted limits

Left ASC Board Position:This parameter displays the running boards position in counts 0-255

Right ASC Board Commanded:This parameter displays the commanded position of the right running board.

Right ASC Board Learned:This parameter will display yes or no to let you know if it has learned it's fully extended and fully retracted limits

Right ASC Board Position:This parameter displays the running boards position in counts 0-255
