Generator Battery Jumper Cable Replacement



GENERATOR BATTERY JUMPER CABLE REPLACEMENT

REMOVAL PROCEDURE




1. Disconnect the negative battery cable.
2. Remove the air cleaner outlet duct.
3. Remove the mega fuse cover.




4. Remove the generator battery jumper cable to mega fuse nut (3).
5. Remove the generator battery jumper cable terminal (4) from the mega fuse stud.
6. Remove the generator battery jumper cable clip (2) from air conditioning (A/C) evaporator tube.
7. Remove the generator battery jumper cable clip (1) from the stud.




8. Reposition the generator battery jumper cable boot (3).
9. Remove the generator battery jumper cable nut (2) from the generator.
10. Remove the generator battery jumper cable terminal (1) from the generator stud.




11. Remove the generator battery jumper cable from the vehicle.

INSTALLATION PROCEDURE




1. Install the generator battery jumper cable to the vehicle.




2. Install the generator battery jumper cable terminal (4) to the mega fuse stud.

3. NOTE: Fastener Notice in Service Precautions.

Install the generator battery jumper cable to mega fuse nut (3).

Tighten the nut to 9 N.m (80 lb in).

4. Install the generator battery jumper cable clip (2) to A/C evaporator tube.
5. Install the generator battery jumper cable clip (1) to the stud.




6. Install the mega fuse cover.




7. Install the generator battery jumper cable terminal (1) to the generator stud.
8. Install the generator battery jumper cable nut (2) to the generator.

Tighten the nut to 9 N.m (80 lb in).

9. Position the generator battery jumper cable boot (3).
10. Install the air cleaner outlet duct.
11. Connect the negative battery cable.
