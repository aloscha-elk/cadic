Electronic Suspension Control Module Programming and Setup



ELECTRONIC SUSPENSION CONTROL MODULE PROGRAMMING AND SETUP

ESC MODULE PROGRAMMING
When the electronic suspension control (ESC) module is replaced, it must be programmed. Refer to Service Programming System (SPS). Programming and Relearning

After programming, the ESC module must have the trimset procedure performed.

To perform the trimset procedure, the vehicle must be sitting at trim height on a level surface. Proceed with the trimset procedure following the onscreen instructions in the special functions menu.
