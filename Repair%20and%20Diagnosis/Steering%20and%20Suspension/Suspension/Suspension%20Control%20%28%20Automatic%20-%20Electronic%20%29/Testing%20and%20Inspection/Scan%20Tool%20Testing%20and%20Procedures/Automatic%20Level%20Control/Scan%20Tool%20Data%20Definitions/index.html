Scan Tool Data Definitions



Scan Tool Data Definitions

Compressor Relay Command

Indicates when the compressor relay is activated.

Controller Battery Volts

Range 10.0-16.0 volts. This data refers to the battery system voltage as measured by the suspension control module.

Exhaust Valve Command

Indicates when the exhaust valve is active.

Position Sensors

Range 0.5-4.85 volts. This data refers to vehicle height measured in volts.

Vehicle speed

Indicates speed of vehicle.
