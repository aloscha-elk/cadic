Inflatable Restraint Sensing and Diagnostic Module Programming and Setup



INFLATABLE RESTRAINT SENSING AND DIAGNOSTIC MODULE PROGRAMMING AND SETUP

IMPORTANT: The air bag indicator light may remain ON after the body control module (BCM), or inflatable restraint sensing and diagnostic module (SDM) is replaced, and during the programming procedure for the BCM until after the procedure is completed. When installing a new SDM, there may be several DTCs set prior to programming. Once programmed, these DTCs should be in history and can be cleared. If both the rollover sensor and the SDM are replaced at the same time, preform Airbag Vehicle Rollover Sensor Programming and Setup first. Inflatable Restraint Vehicle Rollover Sensor Programming and Setup

1. If DTC B0090 is set, refer to DTC B0090. B Code Charts
2. Ignition ON, program the SDM. Service Programming System (SPS)
3. In the BCM Module Setup menu, select Setup SDM Primary Key in BCM. Follow the scan tool on screen instructions.
4. In the SDM Module Setup menu, select setup SDM. Follow the scan tool on screen instructions.
5. cycle the ignition from OFF to ON. Verify no DTCs are set. If any DTCs are set, refer Diagnostic Trouble Codes. A L L Diagnostic Trouble Codes ( DTC )
