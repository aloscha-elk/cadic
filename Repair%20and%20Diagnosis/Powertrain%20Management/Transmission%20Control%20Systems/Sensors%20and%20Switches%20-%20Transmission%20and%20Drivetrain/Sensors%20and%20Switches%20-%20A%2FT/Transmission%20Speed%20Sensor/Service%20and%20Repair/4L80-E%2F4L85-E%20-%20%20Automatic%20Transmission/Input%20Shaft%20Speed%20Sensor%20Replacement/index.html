Input Shaft Speed Sensor Replacement



Input Shaft Speed Sensor Replacement

Removal Procedure

1. Raise and suitably support the vehicle. Refer to Lifting and Jacking the Vehicle.




2. Disconnect the input speed sensor electrical connector (2).




3. Remove the input speed sensor bolt.
4. Remove the input speed sensor (1).
5. Inspect the input speed sensor (1) for any evidence of damage.

Installation Procedure




1. Lubricate the input speed sensor seal with automatic transmission fluid.
2. Install the input speed sensor (1).

Notice: Refer to Fastener Notice.

3. Install the input speed sensor bolt.

Tighten the bolt to 11 N.m (96 lb in).




4. Connect the input speed sensor electrical connector (2).
5. Lower the vehicle.
