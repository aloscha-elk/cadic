Scan Tool Data Definitions



SCAN TOOL DATA DEFINITIONS

Battery Voltage: The scan tool displays the current state of the battery in volts.

Horn Relay Command: The scan tool displays Off/On. The horn output indicates the state that the control module has commanded for the relay.
