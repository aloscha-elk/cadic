Scan Tool Data Definitions



SCAN TOOL DATA DEFINITIONS

The scan tool data definitions contain a brief description of all object detection system related parameters available on the scan tool. The parameters that are available on the scan tool are listed below in alphabetical order.

Battery Voltage: The scan tool displays 0-20 volts. This is the system voltage measured by the object detection module between the battery input and ground.

Calibration Part Number: Displays the object detection module calibration part number.

Component Serial Number 13-16: Displays the object detection module component serial number.

End Model Part Number: The scan tool displays the part number that represents the combination of hardware/software/calibrations present in the object detection module.

Loudspeaker Feed Voltage Output: The scan tool displays the output feed voltage to the radio is measure by volts/counts.

Park Assist Switch: The scan tool displays whether the object detection system is active or inactive.

Park Assist Rear Display LED CKT: The scan tool displays whether the rear display for the object detection system is OK or over current.

Park Assist System Status: The scan tool displays whether the object detection system is enable, disable, inhibit, or failed.

Power Mode: The scan tool displays whether the object detection system is Off, accessory, run, or crank request.

Sensor Feed Voltage Output: The scan tool displays what voltage is being sent to the object sensors.

Vehicle Speed: The scan tool displays the speed of vehicle.

Year Module Built: The scan tool displays the year the object detection module was built.
