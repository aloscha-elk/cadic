Scan Tool Data Definitions



SCAN TOOL DATA DEFINITIONS

Auto Learn: The scan tool displays Enabled when the 30 minute auto learn reprogramming procedure is available as a valid programming option.

Auto Learn Counter: The scan tool displays the count value when using the 30 minute reprogramming procedure. Three 10 minute cycles, or three counts, are required to complete the 30 minute auto learn reprogramming procedure.

Auto Learn Timer: The scan tool displays the time in minutes remaining by the TDM to learn a valid code from the transponder using the 30 minute auto learn reprogramming procedure.

Battery Voltage: The scan tool displays battery voltage at the TDM.

ECM in VTD Fail Enable: The scan tool will display Yes when the ECM is fail enabled.

Ignition Voltage: The scan tool displays ignition voltage at the TDM.

Learned Keys Required: The scan tool displays the numbers of learned keys require to initialize the Adding Keys procedure. Domestic vehicles require 1 key, while export and Canadian vehicles require 2.

Master Keys Learned: The scan tool displays the number of learned master keys.

PCM Auto Learn Timer: The scan tool displays Active when the PCM is in auto learn.

PCM Fail Enable: The scan tool displays Active when the PCM is in the fail enable state.

PCM Fuel Continue: The scan tool displays the status of vehicle fueling. An Inactive value would indicator an invalid or no fuel continue message was received. Vehicle starting is not allowed in this situation. An Active value would indicated a valid fuel continue message was received and vehicle starting is allowed.

PCM Password Learning: The scan tool displays Enabled when the PCM is in learn mode.

PCM Password Status: The scan tool displays the PCM's curent response to the theft password.

Power Mode: The scan tool displays the current vehicle power mode.

Previous TDM State: The scan tool displays the TDM state of the last ignition cycle.

Security Indicator Command: The scan tool displays the state of the BCM commanding the security indicator. The scan tool displays ON or, OFF, as commanded by the BCM.

Tamper Status: The scan tool displays Active when a tamper condition is detected.

Telltale Status: The scan tool displays the state of the body control module (BCM) commanding the security indicator. The scan tool displays ON, OFF, FLASH, or INVALID as commanded by the TDM.

TDM State: The scan tool displays the TDM state for the current ignition cycle.

Total Keys Learned: The scan tool displays the total number of learned keys.

Transponder Key: The scan tool displays Present when a transponder is present. This does not necessarily mean that this is a valid or learned transponder, but only that a transponder is present.

Valid Key: The scan tool displays Yes when a valid transponder key is present.

VTD Fuel Enabled: The scan tool displays Yes when engine starting is allowed.

VTD Fuel Disabled: The scan tool displays Active when engine starting is not allowed.

VTD Fuel Disable Until Ign Off: The scan tool will displays Yes when engine starting is not allowed for this ignition cycle.

VTD Password Learned: The scan tool will display Yes when a valid theft password has been learned.

VTD Auto Learn Timer: The scan too will display Active when auto learn is currently active.
