Transmission Indicators and Messages

The following transmission-related indicators and messages may be displayed on the instrument panel cluster (IPC). For a complete listing and description of all vehicle indicators and messages, refer to Indicator/Warning Message Description and Operation.

4WD

This indicator illuminates when the powertrain control module (PCM) detects that 4WD has been requested.

Tow/Haul

This indicator illuminates when the PCM detects that tow/haul mode has been requested.

Transmission Hot

This message is displayed when the PCM detects a transmission fluid temperature (TFT) equal to or greater than 130° C (266° F) for 5 seconds.

Trans Hot...Idle Engine

This message is displayed when the PCM detects a TFT equal to or greater than 135° C (275° F).
