Input Internal Gear, Reaction Shaft and Shell Installation



Input Internal Gear, Reaction Shaft and Shell Installation

Tools Required

J 36850 Transjel Lubricant




1. Install the reaction sun shell (670) into the reaction sun gear.
2. Install the thrust bearing (669) using J 36850 onto the reaction carrier shaft, tangs up, toward the shaft.
3. Install the input internal gear (664) and reaction carrier shaft (666) assembly into the sun gear shell. Index the reaction carrier shaft spline into the reaction carrier.
