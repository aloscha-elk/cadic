Initial Inspection and Diagnostic Overview



Diagnostic Starting Point - Engine Exhaust

Begin the system diagnosis by reviewing the system Description and Operation. Reviewing the information will help you determine the correct symptom diagnostic procedure when a malfunction exists. It will also help you determine if the condition described by the customer is normal operation.
