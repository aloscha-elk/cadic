Speed Sensor Assembly Removal



Speed Sensor Assembly Removal




1. Remove the input speed sensor bolt (23).
2. Remove the input speed sensor (22A).
3. Remove the output speed sensor bolt (23).
4. Remove the output speed sensor (22B), (output speed sensor plug on 4WD model).
5. Inspect the speed sensors for damage.
