NVG 246-NP8 Transfer Case



DTC C0558

Circuit Description

At each power-up, the transfer case shift control module will check the boot loader for a valid calibration. If no calibration is found the DTC will be set, this DTC will be erased upon programming.

DTC Descriptor

This diagnostic procedure supports the following DTC:

DTC C0558 Calibration Data Not Performed

Conditions for Running the DTC

^ The ignition is ON.
^ The system voltage is 9-18 volts.

Conditions for Setting the DTC

Calibration has not been programmed into the transfer case shift control module.

Action Taken When the DTC Sets

^ The SERVICE 4WD indicator remains illuminated for the remainder of the current ignition cycle.
^ The transfer case control switch does not display the current position LED while DTC C0558 is current.

Conditions for Clearing the DTC

^ Program the transfer case shift control module.
^ Perform the Diagnostic System Check - Vehicle. Initial Inspection and Diagnostic Overview

Test Description

The number below refers to the step number on the diagnostic table.
