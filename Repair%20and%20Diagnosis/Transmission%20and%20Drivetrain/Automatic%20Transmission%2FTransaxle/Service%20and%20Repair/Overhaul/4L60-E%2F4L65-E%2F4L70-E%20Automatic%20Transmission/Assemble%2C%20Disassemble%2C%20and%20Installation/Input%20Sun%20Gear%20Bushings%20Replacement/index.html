Input Sun Gear Bushings Replacement



Input Sun Gear Bushings Replacement

Tools Required
^ J 8092 Universal Driver Handle - 3/4 in - 10
^ J 34196-B Transmission Bushing Service Set

Removal Procedure





Using J 34196-5 which is part of kit J 34196-B with J 8092, remove the input sun gear front bushing (657) and rear bushing (659).

Installation Procedure




1. Using J 34196-4 which is part of kit J 34196-B with J 8092, install the input sun gear front bushing (657).
2. Using J 34196-6 which is part of kit J 34196-B with J 8092, install the input sun gear rear bushing (659).
