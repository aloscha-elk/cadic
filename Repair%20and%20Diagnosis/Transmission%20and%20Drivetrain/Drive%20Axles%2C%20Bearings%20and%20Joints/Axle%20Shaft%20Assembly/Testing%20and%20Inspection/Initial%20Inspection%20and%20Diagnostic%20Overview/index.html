Initial Inspection and Diagnostic Overview



Diagnostic Starting Point - Wheel Drive Shafts

Begin the system diagnosis by reviewing the system Description and Operation. Reviewing the Description and Operation will help you to determine the correct symptom diagnostic procedure when a malfunction exist. Reviewing the Description and Operation information will also help you determine if the condition described by the customer is normal operation. Refer to Symptoms - Wheel Drive Shafts in order to identify the correct procedure for diagnosing the system and where the procedure is located. - Symptoms - Wheel Drive Shafts
