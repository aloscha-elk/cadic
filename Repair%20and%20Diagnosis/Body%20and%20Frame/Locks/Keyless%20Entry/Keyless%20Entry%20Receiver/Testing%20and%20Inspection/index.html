REMOTE CONTROL DOOR LOCK RECEIVER PROGRAMMING AND SETUP

REMOTE CONTROL DOOR LOCK RECEIVER REPLACEMENT
If the remote control door lock receiver is replaced, the following procedures must be performed:
1. Replace remote control door lock receiver (RCDLR) Module.
2. Install Tech 2 and CANDI module.
3. Select Service Programming System and follow on screen instructions. Refer to Service Programming System (SPS). Programming and Relearning
4. With Tech 2 enter Diagnostics and build vehicle with proper Year, Make, Model and Options.
5. Select Body and Accessories.
6. Select Keyless Entry.
7. Select Module Setup, RCDLR.
8. Select Program Key Fobs. Refer to Transmitter Programming. Programming and Relearning
9. Set up a new remote control door lock receiver.
10. Select Tire Type/Pressure selection.
11. Perform Tire Pressure Sensor Learn mode-Refer to Tire Pressure Sensor Learn.
