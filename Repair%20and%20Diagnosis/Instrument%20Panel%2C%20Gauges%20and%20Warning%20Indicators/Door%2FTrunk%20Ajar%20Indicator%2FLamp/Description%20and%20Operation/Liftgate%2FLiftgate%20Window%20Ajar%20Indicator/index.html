Liftgate/Liftgate Window Ajar Indicator



LIFTGATE/LIFTGATE WINDOW AJAR INDICATOR DESCRIPTION AND OPERATION

GATE AJAR INDICATOR
The instrument panel cluster (IPC) illuminates the GATE AJAR indicator when the body control module (BCM) detects that the liftgate or liftglass is ajar or open. The IPC receives a LAN message from the BCM requesting illumination. If the vehicle is in the forward or reverse gear position, the BCM activates an audible warning.
