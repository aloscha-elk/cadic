Diagnostic Starting Point - Tire Pressure Monitoring



Diagnostic Starting Point - Tire Pressure Monitoring

System Familiarization

Review the tire pressure monitor (TPM) system operation in order to familiarize yourself with the correct system functions and operations. Refer to Tire Pressure Monitor Description and Operation.

Visual/Physical Inspection

^ Inspect for aftermarket devices, such as tires, wheels, or radio frequency transmitting devices which could affect the operation of the tire pressure monitor (TPM) system. Refer to Checking Aftermarket Accessories. Component Tests and General Diagnostics
^ Inspect the easily accessible or visible system components for obvious damage which could cause the condition.
^ Inspect the tires for the proper inflation pressures.

System Diagnosis

After incorrect system operation has been verified and that aftermarket devices are not the cause, begin the system diagnosis with the diagnostic system check. the diagnostic system check will provide the following information: Diagnostic System Check - Tire Pressure Monitoring

1. The identification of the control modules that control the TPM system.
2. The ability of those control modules to communicate via the serial data circuit.
3. The presence of any stored diagnostic trouble codes (DTCs) in those control modules and there status.

The use of the diagnostic system check will identify the correct procedure for diagnosing the system and where the procedure is located. Refer to Diagnostic System Check - Vehicle. Initial Inspection and Diagnostic Overview
