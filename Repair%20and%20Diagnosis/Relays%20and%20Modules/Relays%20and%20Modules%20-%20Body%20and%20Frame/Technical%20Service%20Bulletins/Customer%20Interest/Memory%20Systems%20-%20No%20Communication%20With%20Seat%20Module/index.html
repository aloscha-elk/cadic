Memory Systems - No Communication With Seat Module

Bulletin No.: 06-08-50-008

Date: July 26, 2006

INFORMATION

Subject:
Information on Lack of Communication with Memory Seat Control Module After Reprogramming Any Other Control Module

Models:
2007 Cadillac Escalade Models
2007 Chevrolet Suburban Models
2007 GMC Yukon Models

with Individual Front Seats (Non Bucket Seats) (RPO AN3)

Technicians may find that after reprogramming any Electronic Control Unit (ECU) on these vehicles, the Memory Seat Control Module (MSM) may appear to be unable to communicate with the Tech 2(R). However the MSM is fully functional. To restore the ability to communicate with the Tech 2(R), the MSM must be allowed to go to sleep and then awaken. Perform the following steps before attempting to check and clear any Diagnostic Trouble Codes (DTCs) that may have been set as a result of the reprogramming process.

1. After the ECU reprogramming process is complete, rotate the ignition switch to the OFF position.

2. Open the driver's door and wait 60 seconds.

3. Rotate the ignition switch to the RUN position.

4. Clear any DTCs that may have been set during the reprogramming process before returning the vehicle to the customer.

These steps will give the Memory Seat Control Module time to reset and to properly respond to the Tech 2.
