C0775



DTC C0775

Circuit Description

The remote control door lock receiver (RCDLR) receives a radio frequency (RF) transmission from each tire pressure sensor. Each sensor has its own unique identification code contained within its RF transmission. These ID codes must be learned into the RCDLR memory to eliminate all other random RF transmissions from being received by the RCDLR.

DTC Descriptor

This diagnostic procedure supports the following DTC:

DTC C0775 Low Tire Pressure System Sensors Not Programmed/Learned

Conditions for Running the DTC

The ignition is ON.

Conditions for Setting the DTC

No tire pressure sensor ID codes have been learned into the RCDLRs memory.

Action Taken When the DTC Sets

^ DTC C0775 is stored in memory.
^ The driver information center (DIC) displays the SERVICE TIRE MONITOR warning message.

Conditions for Clearing the DTC

^ A current DTC will clear when the sensor ID codes have been learned into the RCDLRs memory.
^ A history DTC will clear after 100 consecutive malfunction free ignition cycles from when the sensor ID codes have been learned into the RCDLRs memory.
^ Using a scan tool.

Diagnostic Aids

A newly replaced RCDLR will set DTC C0775 on its initial ignition ON cycle.
