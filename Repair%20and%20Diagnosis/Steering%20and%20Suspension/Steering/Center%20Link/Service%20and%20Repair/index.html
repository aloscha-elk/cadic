Center Link: Service and Repair



Relay Rod Replacement

Tools Required

J 24319-B Universal Steering Linkage Puller

Removal Procedure




1. Raise the vehicle. Refer to Lifting and Jacking the Vehicle.
2. Remove the engine protection shield, if equipped.
3. Remove the steering damper from the relay rod, if equipped.

Important: Use the proper tool in order to separate all the tie rod and the ball joints.

4. Remove the inner tie rod ends (6).
5. Remove the idler arm ball stud nut (10).
Do not reuse the nut.
6. Remove the pitman arm nut (8).
Do not reuse the nut.




7. Disconnect the idler arm from the relay rod using the J 24319-B.
8. Disconnect the pitman arm from the relay rod using the J 24319-B.
9. Remove the relay rod from the vehicle.
10. Inspect the threads on the tie rod ends for damage.
11. Inspect the ball stud threads for damage.
12. Inspect the ball stud seals for excessive damage.
13. Clean the threads on the ball studs.

Installation Procedure




1. Install the relay rod (9) to the vehicle.
2. Connect the pitman arm ball stud (2) to the relay rod.
3. Connect the idler arm ball stud (1) to the relay rod.
4. Install the new pitman arm prevailing torque nut.

Notice: Refer to Fastener Notice.

5. Install the new idler arm prevailing torque nut.

Tighten the nuts to 62 N.m (46 lb ft).

6. Install the inner tie rod ends.
7. Install the steering damper to the relay rod, if equipped.
8. Install the engine protection shield, if equipped.
9. Lower the vehicle.
10. Check the wheel alignment. Refer to Measuring Wheel Alignment.
