Initial Inspection and Diagnostic Overview



Diagnostic Starting Point - Power Steering System (w/o Electro-Hydraulic Steering)

Begin the system diagnosis by reviewing the system Description and Operation. Reviewing the Description and Operation information will help you determine the correct symptom diagnostic procedure when a malfunction exists. Power Steering System Description and Operation Reviewing the Description and Operation information will also help you determine if the condition described by the customer is normal operation.
